Hábitos saludables y capacitación digital

Para el alumnado

El alumnado dispondrá de diferentes aplicaciones y recursos digitales diseñados para facilitar su aprendizaje y participación en el proyecto. Además de utilizar herramientas para crear contenidos, buscar información y realizar actividades interactivas, contará con enlaces a vídeos explicativos que mostrarán paso a paso cómo trabajar con cada aplicación. Estos materiales servirán de apoyo para favorecer la autonomía, resolver dudas y facilitar el desarrollo de las tareas propuestas. De este modo, se promoverá un aprendizaje accesible, práctico y adaptado al nivel digital del alumnado adulto de TIC.
